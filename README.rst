Kindify
=======

Kindify is a script to transform a single web page to a mobi ebook

Requirements
------------
   * decruft (dcramer version)
   * mobigen or kindlegen
   * wget

Configuration
-------------
Set the DESTDIR in the source code to the dir you want your generated ebook to be created

Usage
-----
kindify <url>
